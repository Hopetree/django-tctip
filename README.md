# django-tctip
一个 Django 应用，用来在页面显示打赏弹框

step 1
```bash
pip install django-tctip
```

step 2
配置中添加应用
```
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'django_tctip',
]
```
添加第三方静态文件路径
